<!-- GETTING STARTED -->
## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/laradelrio/RickAndMorty.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```

### Development server

1. Run
   ```sh
   ng serve
   ```
  2. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
